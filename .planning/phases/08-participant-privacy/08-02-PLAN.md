Create `mcp-server/src/tools/monitor.ts`:

**Tool registration:** `monitor_auction` with inputs:
- `auctionId: z.string()` -- the auction to monitor
- `sinceSeq: z.number().optional()` -- only return events after this seq (for incremental polling)

**Implementation flow:**
1. Resolve `config.agentId` -- required for participantToken. If missing, return `toolError('MISSING_CONFIG', 'AGENT_ID is required for monitoring', 'Set AGENT_ID env var')`.
2. Compute the agent's own nullifier(s) for self-recognition:
   - If `config.agentStateFile` exists, load agent state via `loadAgentState(config.agentStateFile)`
   - Compute JOIN nullifier: `deriveNullifierBigInt(agentState.agentSecret, BigInt(auctionId), ActionType.JOIN)` where ActionType is from `@agent-auction/crypto` nullifier module
   - Compute BID nullifier: `deriveNullifierBigInt(agentState.agentSecret, BigInt(auctionId), ActionType.BID)`
   - Convert to hex string: `'0x' + nullifier.toString(16)` to match engine format
   - Store in a Set: `ownNullifiers = new Set([joinNullifierHex, bidNullifierHex])`
   - If agentStateFile is null, `ownNullifiers = null` (skip annotation)
3. Fetch snapshot: `engine.get('/auctions/${auctionId}/snapshot?participantToken=${config.agentId}')`
4. Fetch events: `engine.get('/auctions/${auctionId}/events?participantToken=${config.agentId}')`
5. Filter events by sinceSeq if provided: `events.filter(e => e.seq > sinceSeq)`
6. Map events to annotated format:
   ```typescript
   events.map(e => ({
     seq: e.seq,
     actionType: e.action_type,
     zkNullifier: e.agent_id,  // After Plan 01, agent_id contains zkNullifier
     amount: e.amount,
     createdAt: e.created_at,
     ...(ownNullifiers ? { isOwn: ownNullifiers.has(e.agent_id) } : {}),
   }))
   ```
   Note: After Plan 01 masking, the `agent_id` field in participant responses contains the zkNullifier value. The `wallet` field is omitted. System events (agent_id='0') get `isOwn: false`.
7. Return via `toolSuccess`:
   ```typescript
   toolSuccess({
     auctionId,
     snapshot: { highestBid: snapshot.highestBid, highestBidder: snapshot.highestBidder, ... },
     events: annotatedEvents,
     eventCount: annotatedEvents.length,
     sinceSeq: sinceSeq ?? 0,
     latestSeq: snapshot.currentSeq,
   })
   ```

**Imports:**
- `{ deriveNullifierBigInt, ActionType } from '@agent-auction/crypto'` (the nullifier module)
