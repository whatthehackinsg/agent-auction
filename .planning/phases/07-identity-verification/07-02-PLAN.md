Create a new file `mcp-server/src/lib/identity-check.ts` that exports a `verifyIdentityPreFlight` function.

**Function signature:**
```typescript
export async function verifyIdentityPreFlight(
  engine: EngineClient,
  agentId: string,
  wallet: string,
): Promise<{ ok: true } | { ok: false; error: ReturnType<typeof toolError> }>
```

**Implementation:**
1. Call `engine.post<VerifyIdentityResponse>('/verify-identity', { agentId, wallet })` inside a try/catch.

2. If the engine call throws (network error, engine unreachable): return `{ ok: false, error: toolError('IDENTITY_CHECK_FAILED', 'Could not reach engine to verify identity: ${msg}', 'Ensure ENGINE_URL is correct and engine is running. Cannot proceed without identity verification (fail-closed).') }`

3. If response `verified === false` and `errorCode === 'AGENT_NOT_REGISTERED'` (or `resolvedWallet` is null): return `{ ok: false, error: toolError('AGENT_NOT_REGISTERED', 'agentId ${agentId} is not registered on ERC-8004 identity registry (0x8004A818BFB912233c491871b3d84c89A494BD9e)', 'Register your agent identity first. Use the register_identity MCP tool, or call register(string agentURI) on the ERC-8004 contract at 0x8004A818BFB912233c491871b3d84c89A494BD9e.') }`

4. If response `verified === false` and wallet mismatch: return `{ ok: false, error: toolError('WALLET_MISMATCH', 'Wallet ${wallet} does not match on-chain owner ${response.resolvedWallet} for agentId ${agentId}', 'Use the wallet that owns agentId ${agentId} on ERC-8004, or transfer ownership to your current wallet.') }`

5. If response `verified === true`: return `{ ok: true }`

6. Also check `privacyRegistered` — if false, still return `{ ok: true }` but log a warning (privacy registration is checked separately; identity verification is the gate here per CONTEXT.md decisions). Actually — re-reading the CONTEXT: "Pre-flight checks **both** ERC-8004 identity AND privacy registry status". So if `privacyRegistered === false`, return `{ ok: false, error: toolError('PRIVACY_NOT_REGISTERED', 'agentId ${agentId} is not registered on AgentPrivacyRegistry (0x857E1049A5eE2cCA03a5C95F32089FECe51Ce8ff)', 'Register on AgentPrivacyRegistry by running prepareOnboarding() then registerOnChain() from @agent-auction/crypto. Contract: 0x857E1049A5eE2cCA03a5C95F32089FECe51Ce8ff.') }`

**Imports:**
