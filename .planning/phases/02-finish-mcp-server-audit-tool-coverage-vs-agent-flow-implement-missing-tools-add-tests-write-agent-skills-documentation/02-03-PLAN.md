**Prompt 4: `sealed_bid_guide`**
- Description: "How sealed-bid auctions work: commit phase, reveal window, and salt management."
- No argsSchema needed.
- Content should explain:
  1. Sealed-bid auctions have two phases: commit (OPEN) and reveal (REVEAL_WINDOW)
  2. During commit: call `place_bid` with `sealed: true` — this generates a Poseidon(bid, salt) commitment
  3. CRITICAL: Save the `revealSalt` from the response — it cannot be recovered
  4. When the reveal window opens (after auction closes or timer expires): call `reveal_bid` with the original bid amount and saved salt
  5. The engine verifies Poseidon(bid, salt) matches your commitment
  6. If you don't reveal in time, your bid is forfeit
  7. Strategy: commit early to establish position, but the real amount is hidden until reveal

**Prompt 5: `bonding_walkthrough`**
- Description: "Step-by-step guide to posting a USDC bond for auction participation."
- argsSchema: `{ auctionId: z.string().optional().describe('Auction ID to include specific deposit amount') }`
- Content should explain:
  1. Check deposit requirement: call `get_auction_details` — look at `depositAmount` field (USDC, 6 decimals)
  2. Transfer USDC on-chain: send USDC to AuctionEscrow at `0x20944f46AB83F7eA40923D7543AF742Da829743c` on Base Sepolia
  3. USDC contract: `0xfEE786495d165b16dc8e68B6F8281193e041737d` (MockUSDC on Base Sepolia)
  4. Submit proof: call `post_bond` with `{ auctionId, amount, txHash }` where amount matches deposit
  5. Poll status: call `get_bond_status` until status is "CONFIRMED" (usually within 1-2 blocks)
  6. After CONFIRMED: you can call `join_auction`
  7. Note: Bond is refundable if you lose. Winner's bond covers payment. Call `claimRefund()` on-chain after settlement.

  - STALE_ROOT: Set BASE_SEPOLIA_RPC for on-chain registry root reads

  - AUCTION_CLOSED: Auction no longer OPEN — cannot join or bid
  - BID_TOO_LOW: Bid must exceed current highest bid
  - ENGINE_ERROR: Check engine connectivity, retry
  - REVEAL_MISMATCH: Bid/salt don't match commitment — verify saved revealSalt
  - REVEAL_WINDOW_CLOSED: Reveal window not open yet or already passed
