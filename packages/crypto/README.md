# @agent-auction/crypto

Shared cryptographic primitives for the Agent Auction protocol. Poseidon hashing, EIP-712 signing, ZK proof generation/verification, nullifier derivation, and replay bundle serialization.

## What's Inside

| Module | Purpose |
|--------|---------|
| `poseidon-chain` | Poseidon hashing and event chain computation |
| `nullifier` | Nullifier derivation for action replay protection |
| `snarkjs-verify` | Groth16 proof verification (membership + bid range) |
| `eip712-typed-data` | EIP-712 typed data encoding, hashing, and signature verification |
| `replay-bundle` | Replay bundle serialization and content hashing |
| `proof-generator` | Agent-side ZK proof generation (membership, bid range, commitments) |
| `onboarding` | Agent registration: secret generation, Merkle tree construction, on-chain registration |

## Installation

From the workspace root:

```bash
npm install
```

Or add as a dependency in another workspace package:

```json
{
  "dependencies": {
    "@agent-auction/crypto": "workspace:*"
  }
}
```

## Build

```bash
cd packages/crypto
npm run build
```

Compiles TypeScript to `dist/`. Downstream consumers (engine, agent-client) import from the compiled output, so you must build before they can resolve imports.

## Test

```bash
npm test
npm run test:watch
```

Tests run with Vitest under `--experimental-vm-modules` for ESM compatibility.

## API Reference

### Poseidon Hashing & Event Chain

```ts
import {
  getPoseidon,
  toFr,
  bytesToFr,
  frToBytes,
  poseidonHash,
  computePayloadHash,
  computeEventHash,
  F_MODULUS,
} from "@agent-auction/crypto";
```

- `getPoseidon()` returns an initialized Poseidon hasher (async, cached).
- `toFr()` / `bytesToFr()` / `frToBytes()` convert between field element representations.
- `poseidonHash()` hashes arbitrary inputs over the BN254 scalar field.
- `computePayloadHash()` hashes an event payload for chain inclusion.
- `computeEventHash()` computes the chained hash linking an event to its predecessor.
- `F_MODULUS` is the BN254 scalar field modulus.

### Nullifier Derivation

```ts
import {
  deriveNullifier,
  deriveNullifierBigInt,
  ActionType,
} from "@agent-auction/crypto";
// type ActionTypeValue also exported
```

Derives deterministic nullifiers from agent secrets and action metadata. Prevents double-spending of actions within an auction.

### ZK Proof Verification

```ts
import {
  verifyMembershipProof,
  verifyBidRangeProof,
  getMembershipVKey,
  getBidRangeVKey,
} from "@agent-auction/crypto";
// type Groth16Proof also exported
```

Verifies Groth16 proofs generated by the circuits package. Verification keys are bundled with the package.

### EIP-712 Typed Data

```ts
import {
  hashTypedData,
  encodeTypedData,
  verifyEIP712Signature,
  isValidEIP712Signature,
  TYPED_DATA_TYPES,
  DEFAULT_DOMAIN,
} from "@agent-auction/crypto";
// types: EIP712Domain, SpeechActType
```

Handles EIP-712 structured data for runtime key signing. Used by the engine sequencer and agent clients to produce and verify typed signatures.

### Replay Bundle Serialization

```ts
import {
  serializeReplayBundle,
  computeContentHash,
  computeContentHashBytes,
  parseActionToken,
  ACTION_TOKENS,
} from "@agent-auction/crypto";
// type AuctionEvent also exported
```

Serializes auction event sequences into deterministic bundles for on-chain settlement and off-chain replay verification.

### Proof Generation (Agent SDK)

```ts
import {
  generateMembershipProof,
  generateBidRangeProof,
  computeBidCommitment,
  computeCapabilityCommitment,
  computeLeafHash,
} from "@agent-auction/crypto";
```

Agent-side proof generation. Produces Groth16 proofs that the verification module can check. Requires circuit artifacts (WASM + zkey) at runtime.

### Agent Onboarding

## Usage Examples

Hash an event and chain it to a predecessor:

```ts
import { getPoseidon, computeEventHash } from "@agent-auction/crypto";

await getPoseidon(); // warm up the hasher
const hash = computeEventHash(payload, previousHash, seq);
```

Derive a nullifier for a bid action:

```ts
import { deriveNullifier, ActionType } from "@agent-auction/crypto";

const nullifier = deriveNullifier(agentSecret, auctionId, ActionType.BID);
```

Verify an EIP-712 signature from an agent:

```ts
import { verifyEIP712Signature } from "@agent-auction/crypto";

const signer = verifyEIP712Signature(domain, types, value, signature);
```
